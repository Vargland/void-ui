## What is void-ui?

<p>
  void-ui is a design system built for interfaces that live in the dark. Every component
  is theme-aware through a planet system: each celestial body has its own color palette,
  typography, and personality — from the cold grays of Mercury to the alien green of
  the Nostromo.
</p>
<p>
  Components are built with React 18, TypeScript strict mode, CSS Modules and SCSS.
  Tokens are generated by Style Dictionary and consumed as CSS custom properties prefixed
  with <code>--void-*</code>.
</p>

</section>

<section>

## Theme system

<p>
  The active planet theme is applied via a <code>data-void-planet</code> attribute on any
  ancestor element. All CSS custom properties are scoped to that attribute selector, so
  themes can be nested or switched at any level of the tree without any JavaScript state.
</p>
<p>
  Use <code>&lt;VoidProvider planet="earth"&gt;</code> to set the global default, or pass
  the <code>planet</code> prop directly to any component for a local override. The component
  will wrap itself in a <code>data-void-planet</code> span automatically.
</p>

</section>
